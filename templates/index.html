Welcome to svg-text-morph!

svg-text-morph is a dynamic SVG animation API built with FastAPI. It lets you generate smooth text morphing animations on the fly.

Check out our GitHub project and remember to give it a star if you like it!

Explore the API documentation at the /docs endpoint.

Ready to generate your animated SVGs? Visit the /generate endpoint.
Need inspiration? What a about preparing a smoothie with some fruits? Try navigating to /generate?words=pineapple,papaya,raspberry and check how it tastes.

Enjoy and happy animating!
